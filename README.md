# Сайт онбординг

## Реализованная функциональность

* План с заданиями на неделю
* Динамическое заполнение шкалы прогресса сотрудника
* Отметки о ходе выполнения заданий на неделю
* Страница с тестом

## Основной стек технологий

* React Jsx
* Css
* Git
* БЭМ
